# Desktop-Assistant

##### Description:
This is a simple desktop assistant with GUI that will allow you control your PC by voice or text (you can read about all features in app by writing "Help me").

#### Usage:
* Python 3.9.7
* dearpygui 0.8.64
* SpeechRecognition 3.8.1
* pyttsx3 2.90
* pycaw 20181226
* keyboard 0.13.5
* comtypes 1.1.10
